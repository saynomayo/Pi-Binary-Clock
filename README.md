# How to Compile

``Make All`` in your terminal

# How to Run

``Make run`` in your terminal

# How to Read

**HH:MM:SS format**

-**Seconds** are to the right in **red**

-**Minutes** are in the middle in **green**

-**Hours** are to the left in **blue**


**Each LED represents a binary bit**
